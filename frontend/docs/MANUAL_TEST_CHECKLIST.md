## 자동 테스트 실행

```bash
cd frontend
npm test -- --testPathPatterns=userFlows
```

**자동 테스트 커버리지**: 플로우 항목 중 데이터/상태/API 상호작용은 `[A]` 마크됨.
시각/애니메이션/실기기 센서(AppState, Native Sharing)는 수동 확인 필요.
